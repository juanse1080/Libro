\chapter{OBJETIVOS}
\section{OBJETIVO GENERAL}
Desarrollar una plataforma software extensible que permita el despliegue, ejecución y visualización de resultados de algoritmos computacionales basados en aprendizaje de máquina y visión por computador.

\section{OBJETIVOS ESPECÍFICOS}
\begin{enumerate}
	\item Identificar los requerimientos de un ambiente de computo capacitado para el despliegue, la ejecución y la visualización de algoritmos de aprendizaje de maquina y visión por computador.
    \item Modelar una arquitectura de software extensible que permita soportar los requerimientos encontrados.
    \item Implementar un prototipo de software extensible para el despliegue, ejecución y visualización de algoritmos de aprendizaje de maquina y visión por computador.
    \item Validar el prototipo de software con 3 algoritmos de aprendizaje de maquina y visión por computador desarrollados por los integrantes del grupo de investigación BivL2ab.
\end{enumerate}
